TrnsysTools.utilities.utilsSpf.isInteger(n)¶

Return True if argument is a whole number, False if argument has a fractional part.

Note that for values very close to an integer, this test breaks. During superficial testing the closest value to zero that evaluated correctly was 9.88131291682e-324. When dividing this number by 10, Python 2.7.1 evaluated the result to zero
